# AWS Masterclass Platform ☁️

An immersive, next-generation learning platform for Amazon Web Services (AWS), designed to make cloud concepts intuitive, interactive, and visually stunning.

## 🔗 Live Demo
[Explore the AWS Masterclass Platform Here](https://MohanAbhishek29.github.io/AWS-Master-Learning-Platform)

## 🚀 Key Features

### 🌌 AWS Galaxy (3D)
Explore the vast universe of AWS services in a fully interactive 3D environment. 
-   **Glowing Orbs & Physics**: Advanced "Anti-Clutter" d3-force physics.
-   **Manual Camera Drift**: Smooth, cinematic auto-rotation to explore Compute, Storage, Database, Messaging hubs.
-   **Size Hierarchy**: Visual weights proportional to service importance (e.g., AWS Core > Hubs > Leaves).

### 🤖 AI Cloud Assistant
An immersive, full-screen interactive AI companion.
-   **Context-Aware**: Ask questions, request architectures, or get study tips.
-   **Dynamic UI**: Floating chat interface with glassmorphism and real-time response streams.

### 🏗️ Architecture Builder
A powerful drag-and-drop canvas to design and visualize cloud architectures.
-   **Drag & Drop**: Easily place EC2, S3, RDS, and other nodes.
-   **Smart Connections**: Link services with animated data flow lines.
-   **Zoom & Pan**: Professional-grade canvas controls with a "Scan" fit-view.

### 💻 CLI Hacker Missions
Step into the terminal and solve real-world AWS scenarios in a gamified CLI environment.
-   **Immersive Terminal**: Type commands, decrypt clues, and disable rogue AI cores.
-   **Timed Challenges**: Work under pressure with countdown timers and ticking heartbeat audio.

### ⚔️ The Arena (Gamified Learning)
Test your knowledge in a high-stakes competitive environment.
-   **Practice Mode**: Endless learning with instant feedback.
-   **Time Attacks**: "Blitz" and "Marathon" modes.
-   **Streak System**: Earn multipliers and unlock ranks.

### � Region Globe
Interactive 3D visualization of AWS Global Infrastructure. Spin the globe to explore Regions and Availability Zones.

### �🏘️ Community Hub
A vibrant ecosystem for cloud architects to share and compete.
-   **Blueprint Gallery**: 25+ high-resolution AWS schematics across AI, Security, IoT, and Serverless.
-   **Hall of Fame**: Interactive leaderboards for top engineers with XP tracking and podium animations.
-   **Live Pulse**: Real-time activity feed tracking code merges, rank-ups, and badge unlocks.
-   **High-Res Downloads**: Export any blueprint schematic as a stylized PNG for professional documentation.

###  Global Search
Type-ahead search functionality to instantly find any service, cheat sheet, or learning path.

## 🛠️ Technology Stack

*   **Frontend**: React 18, Vite
*   **Styling**: Vanilla CSS (Variables, Glassmorphism, Neon Effects)
*   **Animations**: Framer Motion
*   **3D Graphics**: Three.js, React Three Fiber (R3F), Drei
*   **Diagramming**: React Flow
*   **Icons**: Lucide React

## 💻 How to Run (Development Mode)

This project uses a modern build tool (Vite) and **must be run on a local server**.

1.  **Open Terminal** in the project folder.
2.  **Install Dependencies** (First time only):
    ```bash
    npm install
    ```
3.  **Start the App**:
    ```bash
    npm run dev
    ```
4.  **Open in Browser**:
    Click the link shown in the terminal (usually `http://localhost:3000`).

## 🎨 Design Philosophy
*   **Dark Mode First**: Optimized for developer focus and visual comfort.
*   **Glassmorphism**: Translucent panels and blur effects for depth.
*   **Motion**: Subtle micro-interactions and smooth transitions to keep the interface "alive".
